Proyecto óperas. Práctica de php, jquery y sql

Partiendo de una base de datos con dos tablas relacionadas entre si, se llega al siguiente ejemplo.

Por un lado tenemos el enlace de autores, en el que se nos muestra información básica del autor y sus obras. Los autres se obtienen directamente desde la base de datos y se muestran en un input de tipo selecionable.

Por otro lado tenemo el enlace a las óperas que tenemos guardadas en la base de datos. En este caso se nos muestra informacion relacionada con la obra.
